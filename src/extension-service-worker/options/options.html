Blockchain providers

Chrome will use these providers to access blockchain data. These providers will be able to record your web3 browsing history, similar to how your ISP may monitor your network connection. You can use default public providers, or run your own nodes to access onchain websites verifiably and more privately. If multiple providers are configured for the same chain, your browser will query all providers in parallel and use the first response.
